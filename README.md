# AltQ v7.2.6

## Quick Start
1. Open Chrome and go to **chrome://extensions**
2. Enable **Developer mode**
3. Click **Load unpacked** and select **AltQ-v7.2.6**

## Usage
- Select any text to see the AltQ popup
- Press **Alt + Q** to send text to the last AI instantly
- Alt + W returns you to the source tab (fallback also works even if command is not assigned)
- Optional: create up to 3 Custom Prompts (stored locally)
- Note: You can add up to 3 custom services (HTTPS only)

## What's New in 7.2.6
- Prompt formatting normalized (prompt + blank line + selected text)
- Time Saved analytics (25s simple / 150s template)
- Return-to-source hotkey fallback (Alt+W)

## Troubleshooting
- Popup missing? Refresh the page or re-load the extension
- PDFs / Google Docs: enable "Allow access to file URLs" in chrome://extensions
- Toolbar icon or gear always opens options.html for manual tests
